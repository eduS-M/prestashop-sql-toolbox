# PrestaShop SQL Toolbox 🚀

Colección de consultas SQL para optimizar la gestión de PrestaShop.

## 📋 Contenido

### Clientes
* **Clientes sin compras:** Identifica usuarios registrados que no han generado pedidos para campañas de retargeting.
* **Segmentación por grupos:** Conteo de miembros por grupo de cliente.

### Pedidos
* **Revisión de Pedidos:** Comparativa entre el registro físico y sistémico para detectar descuadres.
* **Transportistas:** Mapeo de zonas, comunas y regiones por transportista.

## ⚠️ Notas
* La mayoría de las consultas usan el prefijo de tabla predeterminado `ps_`. 
* Algunas consultas de clientes filtran IDs específicos de administración (ej. 1323, 10).
